# List of EJS/HTML Pages for Property Management Website

## Public Pages
- login.ejs — Page where users can log in (not functional yet)

## Logged-in Pages
- landingpage.ejs — Home page after logging in; shows quick links and recent info

### Units
- units_list.ejs — Shows list of properties owned by the user
- unit_detail.ejs — Shows information about a specific unit (tenant info, etc.)

### Maintenance
- maintenance_list.ejs — Shows all maintenance requests (pending and completed)
- maintenance_new.ejs — Page to create a new maintenance request
- maintenance_detail.ejs — Shows details about a single maintenance request

### Expenses
- expenses_month.ejs — Shows monthly utilities and expenses
- expense_new.ejs — Page to add a new expense

### Message Board
- board.ejs — Message board page for owners to post and reply
- thread_detail.ejs — Shows a single discussion thread with replies

### Calendar
- calendar.ejs — Shared calendar to view and add meetings/events

### Account
- profile.ejs — User profile page (name, email, etc.)
